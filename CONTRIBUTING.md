# Contributing to Wireframe.ai.ai

Thank you for your interest in contributing to Wireframe.ai.ai, the open-source AI-powered Web Design Agent by Designbyte Studio! We welcome all contributions, from bug reports and feature requests to code implementation and documentation improvements.

## How to Contribute

### 1. Reporting Bugs
- Describe the bug clearly.
- Include steps to reproduce the issue.
- Provide any relevant errors, screenshots, or code snippets.

### 2. Suggesting Enhancements
- Explain your feature idea.
- Provide a clear use case for why the feature would be beneficial.
- Optionally, describe how you think it could be implemented.

### 3. Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Commit your changes with descriptive messages:
   ```bash
   git commit -m "Add some feature"
   ```
4. Push your branch to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
5. Open a Pull Request from your branch to the main repository. Follow the PR template guidelines if provided.

## Development Setup

Please refer to the `README.md` for full instructions on setting up your environment locally.

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the development server:
   ```bash
   pnpm dev
   ```
